# Alura-PortoSeguro

Repository for Alura-PortoSeguro course labs and exercises.

## Structure

This repository contains Jupyter notebook labs organized in the `labs/` directory:

- **Lab 01**: Introduction and basic concepts
- **Lab 02**: Intermediate exercises
- **Lab 03**: Advanced topics

## Requirements

- Python 3.x
- Jupyter Notebook or JupyterLab

## Usage

1. Clone this repository
2. Install Jupyter: `pip install jupyter`
3. Navigate to the labs directory: `cd labs/`
4. Start Jupyter: `jupyter notebook`
5. Open any lab notebook to begin
